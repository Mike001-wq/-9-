Usage:

For example:

```python
import pickle
dataset = ['kishida_speech', 'yasushi_speech', 'hayashi_speech']
dataset_pk_path = f"{dataset[2]}.pk"
dataset_pk = open(dataset_pk_path, 'rb')
speech = pickle.load(dataset_pk)
dataset_pk.close()
print(speech)
```
